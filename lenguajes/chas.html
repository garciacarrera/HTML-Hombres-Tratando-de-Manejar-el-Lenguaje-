C#

Características

    Tipo de usos

  1. C# es ideal para el desarrollo de aplicaciones de Windows, aplicaciones móviles, desarrollo de juegos, aplicaciones web, aplicaciones empresariales, automatización y tareas del sistema, aplicaciones de escritorio multiplataforma, y desarrollo de servicios y APIs

    2. Paradigmas

  2. En resumen, C# es un lenguaje de programación multiparadigma que combina la orientación a objetos con capacidades de programación funcional, imperativa, declarativa y concurrente, ofreciendo una amplia gama de herramientas y técnicas para desarrollar software de manera eficiente y flexible

    Historia

  • C#, pronunciado como “C Sharp,” es un lenguaje de programación desarrollado por Microsoft y se ha convertido en una parte fundamental del ecosistema de desarrollo de software de la empresa. Su historia comienza a finales de la década de 1990 cuando Microsoft buscaba crear un lenguaje moderno y versátil que pudiera competir con otras tecnologías de programación.

    • Tipado

  • C# es un lenguaje fuertemente tipado. Todas las variables y constantes tienen un tipo, al igual que todas las expresiones que se evalúan como un valor. Cada declaración del método especifica un nombre, el tipo y naturaleza (valor, referencia o salida) para cada parámetro de entrada y para el valor devuelto

    • Nivel de lenguaje

  • Es un lenguaje de programación multiparadigma desarrollado por Microsoft, que evoluciona de la familia de lenguajes C -como su nombre indica- tomando lo mejor de los lenguajes C y C++ y que se asemeja mucho a lenguajes de alto nivel de abstracción como Java y JavaScript.
