JavaScript

Características

    Tipo de usos

  1. Su uso es orientado a objetos y principalmente para desarrollo web front-end y back-end, ya que ofrece la creación de interfaces de usuario interactivas y dinámicas. Del lado back-end se usa para desarrollar parte de los servidores de apps web con ayuda de Node.js. Otros usos incluyen el desarrollo móvil en conjunto con el framework React Native, también en Ciencia de datos y Juegos web.

    2. Paradigmas

  2. Su tipo de paradigma es Imperativo, centrándose en las instrucciones paso a paso para modificar el estado del programa.

    Historia

  • Este lenguaje fue creado por Brendan Eich en 1995, diseñado para añadir elementos dinámicos e interactivos a páginas web estáticas en HTML. Fue implementado por primera vez en "Netscape Navigator", el navegador más conocido de esa época, para luego ser rápidamente adoptado por Microsoft para Internet Explorer.

    • Tipado

  • Es de tipo débilmente tipado, lo que quiere decir que las variables no tienen un tipo de dato asociado de manera explícita; este mismo se determina en tiempo de ejecución.

    • Nivel de lenguaje

  • Es de alto nivel porque abstrae detalles de la arquitectura de la computadora y permite que los programadores se centren en la lógica del programa. También proporciona una sintaxis parecida al inglés, siendo más fácil su lectura y escritura.
