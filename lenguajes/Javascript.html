JavaScript

Características

    Tipo de usos

  1. Su uso es orientado a objetos y principalmente para desarrollo web front-end y back-end, ya que ofrece la creación de interfaces de usuario interactivas y dinámicas. Del lado back-end se usa para desarrollar parte de los servidores de apps web con ayuda de Node.js. Otros usos incluyen el desarrollo móvil en conjunto con el framework React Native, también en Ciencia de datos y Juegos web.

    2. Paradigmas

  2. Se caracteriza principalmente por ser un lenguaje de programación orientado a objetos (POO) polivalente y fácil de aprender. La POO se basa en estructurar un programa en piezas simples y reutilizables llamadas clases para crear instancias individuales a partir de estas, también conocidas como objetos. Esto permite dejar de centrarnos en la lógica pura de los programas y empezar a pensar en objetos y las relaciones o interacciones que existen entre ellos.

    Historia

  • C#, pronunciado como “C Sharp,” es un lenguaje de programación desarrollado por Microsoft y se ha convertido en una parte fundamental del ecosistema de desarrollo de software de la empresa. Su historia comienza a finales de la década de 1990 cuando Microsoft buscaba crear un lenguaje moderno y versátil que pudiera competir con otras tecnologías de programación.

    • Tipado

  • C# es un lenguaje fuertemente tipado. Todas las variables y constantes tienen un tipo, al igual que todas las expresiones que se evalúan como un valor. Cada declaración del método especifica un nombre, el tipo y naturaleza (valor, referencia o salida) para cada parámetro de entrada y para el valor devuelto.

    • Nivel de lenguaje

  • Es un lenguaje de programación multiparadigma desarrollado por Microsoft, que evoluciona de la familia de lenguajes C, tomando lo mejor de los lenguajes C y C++ y asemejándose mucho a lenguajes de alto nivel de abstracción como Java y JavaScript. Forma parte de la plataforma .NET de Microsoft, una API que facilita la construcción de aplicaciones multiplataforma sólidas y duraderas. Sin perder la potencia original de C, permite acceder a bajo nivel al núcleo de los sistemas operativos, trabajar con punteros a memoria e interactuar con elementos físicos de los dispositivos.
