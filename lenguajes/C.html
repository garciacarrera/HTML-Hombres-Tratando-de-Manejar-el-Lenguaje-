C

Características

    Tipo de usos

  1. Optimización de código en base a C++ Protección de información Escritura de código en modo kernel Desarrollo de software integrado

    2. Paradigmas

  2. C es un lenguaje del paradigma imperativo y estructurado, que facilita la programación modular permitiendo la creación de unidades que pueden compilarse de forma independiente y luego enlazarse.

    Historia

  • C es un lenguaje de programación de propósito general desarrollado por Dennis Ritchie entre 1969 y 1972 en los Laboratorios Bell, evolucionando del lenguaje B, el cual estaba basado en BCPL. Fue diseñado inicialmente para la implementación de sistemas operativos, especialmente Unix.

    • Tipado

  • C es un lenguaje de tipos de datos estáticos y débilmente tipado.

    • Nivel de lenguaje

  • C es considerado un lenguaje de nivel medio, ya que combina características de lenguajes de alto nivel con la capacidad de control de bajo nivel.
